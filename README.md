Tarea 4.1 - Formulario Gráfico con Clase de Procesamiento Materia: Programación 1 Descripción: Este proyecto implementa una aplicación en Java con una interfaz gráfica (GUI) desarrollada en Netbeans, la misma permite capturar y presentar los datos de un participante aplicando principios como:
•	Encapsulamiento
•	Separación de responsabilidades
•	Uso de getters y setters
•	Comunicación entre GUI y clase de procesamiento
Los datos que ingresa el usuario, se almacenan mediante la clase ProcesarDatos, y luego pueden mostrarse tanto en consola como en una ventana emergente.
